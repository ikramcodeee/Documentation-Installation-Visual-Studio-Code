# Cara Install Visual Studio Code di Windows untuk Pemula

> **Visual Studio Code** adalah code editor gratis yang ringan, cepat, dan kaya fitur. Panduan ini akan memandu kamu melalui proses instalasi VS Code di Windows dari awal hingga siap digunakan.

---

## Apa itu Visual Studio Code?

Visual Studio Code (VS Code) adalah code editor buatan Microsoft yang bisa digunakan untuk menulis kode dalam berbagai bahasa pemrograman seperti Python, JavaScript, HTML, CSS, dan banyak lagi. VS Code sangat populer di kalangan developer karena:

- **Gratis dan open source**
- **Ringan** — tidak memakan banyak resource komputer
- **Banyak ekstensi** — bisa dikustomisasi sesuai kebutuhan
- **Tersedia di Windows, macOS, dan Linux**

---

## Prasyarat

Sebelum mulai, pastikan komputer kamu memenuhi kebutuhan berikut:

- Sistem operasi: **Windows 10 atau Windows 11**
- Ruang penyimpanan kosong: minimal **376 MB**
- Koneksi internet yang stabil

---

## Langkah-Langkah Instalasi

### 1. Buka Website Resmi VS Code

Buka browser yang kamu gunakan, lalu kunjungi:

```
https://code.visualstudio.com
```

Setelah halaman terbuka, klik tombol **"Download for Windows"** untuk mulai mengunduh file instalasi.

> 💡 **Tips:** Pastikan kamu mengunduh dari situs resmi untuk menghindari file yang tidak aman.

---

### 2. Tunggu Proses Download Selesai

Setelah tombol download diklik, proses pengunduhan akan berjalan secara otomatis. File yang sedang diunduh dapat dilihat di riwayat download browser-mu.

Jika download tidak dimulai secara otomatis, gunakan tautan **"Direct Download Link"** yang tersedia di halaman yang sama.

File yang diunduh akan bernama kurang lebih seperti ini:

```
VSCodeUserSetup-x64-1.xx.x.exe
```

---

### 3. Jalankan File Instalasi

Setelah download selesai, buka file `.exe` yang ada di folder **Downloads** pada komputermu untuk memulai proses instalasi.

---

### 4. Setujui Syarat dan Ketentuan

Pada layar pertama, kamu akan melihat **License Agreement** (syarat dan ketentuan penggunaan). Lakukan langkah berikut:

1. Baca syarat dan ketentuan yang ditampilkan
2. Centang pilihan **"I accept the agreement"**
3. Klik tombol **"Next"** untuk melanjutkan

---

### 5. Tentukan Lokasi Instalasi

Di tahap ini, kamu bisa memilih di mana VS Code akan disimpan di komputermu.

- Jika tidak ingin mengubah lokasi, biarkan saja lokasi default yang sudah tersedia
- Jika ingin menyimpan di tempat lain, klik **"Browse"** dan pilih folder yang diinginkan

Setelah selesai, klik **"Next"**.

---

### 6. Pilih Folder Menu Start

Selanjutnya, tentukan nama folder untuk shortcut VS Code di **Start Menu** Windows. Kamu bisa menggunakan nama default _"Visual Studio Code"_ atau menggantinya sesuai keinginan.

Klik **"Next"** untuk melanjutkan.

---

### 7. Pilih Opsi Tambahan

Di halaman ini tersedia beberapa opsi tambahan. Berikut rekomendasinya:

| Opsi                                             | Rekomendasi | Keterangan                                     |
| ------------------------------------------------ | ----------- | ---------------------------------------------- |
| Create a desktop icon                            | ✅ Aktifkan | Shortcut di desktop untuk akses cepat          |
| Add "Open with Code" to file context menu        | ✅ Aktifkan | Klik kanan file langsung buka di VS Code       |
| Add "Open with Code" to directory context menu   | ✅ Aktifkan | Klik kanan folder langsung buka di VS Code     |
| Register Code as editor for supported file types | ✅ Aktifkan | VS Code jadi editor default                    |
| Add to PATH                                      | ✅ Aktifkan | Memungkinkan menjalankan VS Code dari terminal |

Setelah memilih opsi yang diinginkan, klik **"Next"**.

---

### 8. Mulai Instalasi

Kamu akan melihat ringkasan semua pengaturan yang sudah dipilih. Pastikan semuanya sudah sesuai, lalu klik tombol **"Install"** untuk memulai proses pemasangan.

Tunggu beberapa saat hingga progress bar selesai.

---

### 9. Instalasi Selesai

Setelah proses selesai, akan muncul layar konfirmasi. Centang opsi **"Launch Visual Studio Code"** jika ingin langsung membuka VS Code, lalu klik **"Finish"**.

🎉 **Selamat! Visual Studio Code sudah berhasil terpasang di komputermu.**

---

## Langkah Selanjutnya

Setelah VS Code terpasang, berikut beberapa hal yang bisa kamu lakukan:

- **Pilih tema tampilan** — Buka `File > Preferences > Color Theme` untuk mengubah tema editor
- **Install ekstensi** — Klik ikon Extensions di sidebar atau tekan `Ctrl+Shift+X`
- **Buka folder project** — Gunakan `File > Open Folder` untuk mulai bekerja dengan project-mu
- **Coba terminal terintegrasi** — Tekan `Ctrl+`` (backtick) untuk membuka terminal langsung di dalam VS Code

---

## Troubleshooting

**Instalasi gagal atau error?**

- Pastikan kamu memiliki hak akses administrator di komputer
- Coba jalankan file installer dengan klik kanan → _"Run as administrator"_

**Download tidak bisa dimulai?**

- Coba gunakan browser lain (Chrome, Firefox, atau Edge)
- Gunakan "Direct Download Link" yang tersedia di halaman resmi VS Code

**VS Code tidak muncul di Start Menu?**

- Coba restart komputer setelah instalasi selesai
